# IMAGE-TO-EMOJI
# 😀 EmojiVerse Studio

> **Transform your photos into creative emoji-based artworks directly in your browser.**

EmojiVerse Studio is a modern web application built with **HTML5, CSS3, and Vanilla JavaScript** that allows users to upload images and convert them into fun emoji creations. The project combines browser-based image processing, Canvas API, and face detection technologies to provide an interactive and visually appealing editing experience.

## ✨ Features

### 😀 Emoji Mosaic

* Convert images into emoji mosaics.
* Smart emoji selection based on image colors.
* Adjustable emoji size and density.
* Multiple emoji categories.
* Live preview.
* Zoom support.
* Download as PNG.
* Before & After comparison.

---

### 🟨 Pixel Emoji Art

* Convert images into pixel-style emoji artwork.
* Adjustable pixel size.
* Brightness and contrast controls.
* Saturation adjustment.
* Grayscale mode.
* Invert colors.
* Random emoji mode.
* High-quality export.

---

### 😎 Emoji Face Filters

* Browser-based face detection.
* Detect facial landmarks.
* Apply fun emoji filters.
* Resize, rotate and move filters.
* Stack multiple filters.
* Capture snapshots.
* Download edited images.

Available Filters:

* 😎 Sunglasses
* 😂 Laugh
* ❤️ Heart Eyes
* 🤖 Robot
* 👻 Ghost
* 🐼 Panda
* 🐱 Cat
* 🐶 Dog
* 👑 Crown
* 🎧 Headphones
* 🌸 Flower
* ✨ Sparkles

---

### 🧑 Cartoon Emoji Avatar

Generate a customizable cartoon avatar from uploaded photos.

Customization Options:

* Hair styles
* Eye styles
* Mouth expressions
* Accessories
* Background themes
* Skin tone adjustments

Export avatar as:

* PNG
* SVG
